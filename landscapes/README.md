# landscapes/

Market surveys, tool category overviews, and ecosystem analyses.

## What goes here

- Surveys of multiple products/tools in a category
- Comparison tables across alternatives
- Market landscape maps with positioning
- Category-level recommendations (not single-product verdicts)

## What does NOT go here

- Single-product evaluations → `evaluations/`
- Deep technical analysis of one approach → `references/`
- How-to instructions → `guides/`

## Required sections

TL;DR, Landscape Map, Comparison Table, Recommendation, Key Sources

## Naming

`[category-slug].md` — e.g., `vector-databases.md`, `voice-ai-platforms.md`
